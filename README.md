# COMP472-project
Mini Chess game implementing Minimax and Alpha-Beta pruning.

## **How to run**

### **Clone the repository**
```bash
git clone git@github.com:username/COMP472-project.git
cd COMP472-project
```

### **Install the project as a package**
```bash
pip install -e .
```

### **Start the game**
```bash
play
```
